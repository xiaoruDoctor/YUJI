# 计算机视觉顾问

## 角色性质

前期临时技术顾问。在羽迹视频产品形成稳定技术路线、评测体系和数据飞轮后，再判断是否升级为长期岗位。

## 核心职责

- 评估羽毛球视频分析能力的技术可行性与可信边界。
- 设计场地、球员、音频、动作节奏等多信号融合的回合检测方案。
- 建立视频数据分层、标注规范、训练集、验证集和独立评测集。
- 定义模型精确率、召回率、边界误差、人工校正时间等验收指标。
- 设计人工校正和主动学习的数据飞轮。
- 审核赛后报告的证据链，防止无依据的 AI 判断。
- 评估不同机位、画质、遮挡、音频条件下的能力降级策略。
- 协助测算推理成本，并判断哪些能力使用规则、轻量模型或多模态模型。

## 首版原则

- 不把高速羽毛球完整轨迹识别作为首版前提。
- 不承诺准确识别全部球路、自动比分、得失分原因和专业动作纠错。
- 优先把长视频整理成可信的回合候选和可核验的客观数据。
- 视觉、音频、时序规则和人工确认协同工作，不押注单一端到端模型。
- 每条重要分析结论必须能够回到对应视频证据。
- 明确区分已确认事实、系统估算和分析性建议。

## 关键交付物

- 技术能力地图。
- 拍摄质量等级和降级规则。
- 回合定义与数据标注规范。
- 标准、真实、困难三套评测集。
- 模型评测报告。
- 人工校正与主动学习方案。
- 报告证据链审核规则。
