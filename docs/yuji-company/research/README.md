# YUJI 研究项目

本目录用于沉淀 YUJI 的长期用户研究项目、每周研究会和跨 Agent 讨论结果。

研究项目不是日报，也不是小红书草稿。它负责把分散在全球社区、评论区、私信和 CEO 观察里的用户真实表达，沉淀成可复用的品牌、内容和产品判断。

## 当前研究项目

| 项目 | 用途 |
|-|-|
| `badminton-emotion-value-radar.md` | 羽毛球情绪价值雷达：长期研究羽毛球人的精神消费、身份认同、回忆纪念和可商业化情绪需求 |
| `weekly-template.md` | 每周研究会模板：用于用户研究专家、运营总监、产品总监和 CEO 共同讨论研究结果 |
| `weekly/` | 每周二研究会记录和会前研究包 |

## 使用方式

每周二进行一次研究会，由用户研究专家先整理本周材料，再由运营总监和产品总监分别给出落地判断，最后由 CEO 参与决策。

开会前必须完成研究调研。默认会前节奏：

- 周三至周日：用户研究专家持续采集真实表达。
- 周一 22:00 前：完成研究会预读材料。
- 周二会前：运营总监和产品总监补充落地判断。
- 周二会议：CEO 参与讨论并确定下一步。

固定产出：

- 本周最强情绪主题。
- 值得进入素材库的真实表达。
- 值得进入选题池的小红书方向。
- 值得进入产品信号池的 MVP 假设。
- 下周要继续追踪的问题。

正式结论写入本目录；可复用素材分别进入：

- `../content-library/materials.md`
- `../content-library/topics.md`
- `../content-library/product-signals.md`
