# 小羽衣橱库

本文件用于管理小羽在小红书、视频、羽迹卡样稿和品牌物料里的服装选择。

小羽可以在不同笔记中穿不同衣服，避免每次发布都像同一套模板图。但小羽不是运动穿搭博主，服装只服务羽毛球情绪、场景真实感和小红书停留感，不能抢走内容主角。

## 核心规则

- 不同笔记可以更换服装。
- 同一篇图文、同一组视频或同一张羽迹卡系列中，小羽必须保持同一套服装和主要道具。
- 每次发布前优先选择最近 14 天未使用过的服装。
- 每次选中服装后，要把服装 ID 写入对应笔记或资产记录。
- 服装必须适合真实羽毛球运动，不能只为好看而牺牲运动合理性。
- 运动裙裤必须写明有安全裤。
- 不做低胸、湿身、写真、擦边、刻意腿部特写、夸张裙摆、过度甜腻穿搭或不适合打球的衣服。
- 前台正文不讲衣服、不讲穿搭，除非内容主题本身就是女生球友状态。

## 选择方式

默认使用 `14 天不重样` 规则：

1. 写小羽笔记或生图提示词前，先看本文件的“使用记录”。
2. 从最近 14 天未使用过的服装里选择一套。
3. 如果没有特殊场景要求，可以随机选择。
4. 如果内容场景很明确，按场景优先选择：
   - 下班到球馆：优先 `XY-OUTFIT-04`、`XY-OUTFIT-06`、`XY-OUTFIT-09`、`XY-OUTFIT-12`
   - 场边休息：优先 `XY-OUTFIT-02`、`XY-OUTFIT-05`、`XY-OUTFIT-10`、`XY-OUTFIT-14`
   - 准备上场：优先 `XY-OUTFIT-01`、`XY-OUTFIT-03`、`XY-OUTFIT-07`、`XY-OUTFIT-11`
   - 动态接发：优先 `XY-OUTFIT-03`、`XY-OUTFIT-08`、`XY-OUTFIT-13`
   - 最后一局 / 散场：优先 `XY-OUTFIT-06`、`XY-OUTFIT-09`、`XY-OUTFIT-12`
5. 选中后，把完整服装提示词片段复制进每一条生图提示词，不能只写“参考衣橱库第几套”。

随机选择可用命令：

```bash
seq -w 1 14 | shuf -n 1
```

得到编号后选择对应 `XY-OUTFIT-编号`。如果该编号最近 14 天用过，就重新随机。
