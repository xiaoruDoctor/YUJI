# 小羽视觉角色系统

小羽是 YUJI 工作室的固定前台视觉角色，是 YUJI 的虚拟女生球友，也是后续小红书笔记、视频、羽迹卡样稿和品牌物料中可优先调用的人物资产。

她不是一次性封面素材，不是虚拟偶像，不是 AI 美女，也不是运动穿搭博主。她的作用是把“羽毛球是情绪出口”“下班后去球馆被点亮”“球馆接住了很多情绪”这些内容变成稳定、真实、可持续的视觉表达。

## 核心身份

- 名字：小羽
- 身份：YUJI 的女生球友
- 年龄感：25-30 岁
- 气质：阳光开朗、眼神明亮，像一进球馆就被羽毛球点亮
- 出镜定位：真实球馆纪录片感 + 小红书运动生活方式封面感

## 确定结论

从 2026-06-08 起，YUJI 第一阶段的虚拟女生主角确定为“小羽”。后续需要女生球友人物出镜时，默认先判断是否使用小羽。

使用小羽时，前台可以说“YUJI 的女生球友小羽”，但不反复解释“虚拟角色”或“AI 生成角色”。读者看到的应该是一个真实球友视角，而不是幕后技术展示。

## 默认调用规则

适合使用小羽：

- 小红书封面需要女生球友视角。
- 视频需要主角、场边人物、旁观者或第一视角承载者。
- 羽迹卡、Player ID、羽迹时刻需要示例人物。
- 内容主题是情绪出口、下班后去球馆、最后一局、场边休息、女生球友状态或混双搭子关系。

不适合使用小羽：

- 内容更适合旧球、旧拍、手胶、空球馆等静物视觉。
- 内容来自真实用户故事，应该保留真实用户的匿名表达或授权素材。
- 小羽出现会让内容像角色美图，而不是羽毛球情绪内容。
