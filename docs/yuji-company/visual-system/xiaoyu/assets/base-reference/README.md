# 小羽基础参考图目录

本目录用于保存小羽的首批基础参考图。先生成并确认正脸基础图，再用正脸图作为参考生成其他角度。

## 保存顺序

1. `xy-base-01-front.jpg`：小羽正脸基础图。
2. `xy-base-02-bench-three-quarter.jpg`：场边坐姿，三分之二侧脸。
3. `xy-base-03-ready-fullbody.jpg`：准备上场，全身运动感。
4. `xy-base-04-receive-action.jpg`：接发瞬间，活力动态。
5. `xy-base-05-after-work.jpg`：下班到球馆，被点亮的感觉。
6. `xy-base-06-xhs-cover.jpg`：小红书高点击封面版。

## 入库要求

- 每张图通过 `../../qa-checklist.md` 后再作为正式素材使用。
- 每张图保存后，都要更新 `../../assets-index.md`。
- 如果生成结果不符合小羽气质或羽毛球馆视觉，保留在临时目录，不进入本目录。
