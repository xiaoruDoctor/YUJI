# 小羽视觉质量检查清单

每张小羽图片或视频截图进入正式内容前，必须按本清单检查。只要有严重失败项，就不能进入小红书、视频或品牌物料。

## 角色一致性

- [ ] 看起来是同一个小羽，而不是随机亚洲女生。
- [ ] 脸型、五官、发色和气质与正脸基础图一致。
- [ ] 年龄感在 25-30 岁之间，是成年人。
- [ ] 眼神明亮，笑容自然，有被球馆点亮的状态。
- [ ] 没有郁闷、冷脸、丧感、忧伤或过度营业笑。

## 羽毛球真实感

- [ ] 场景是真实羽毛球馆，不是篮球馆、排球馆或综合馆错觉。
- [ ] 地面是常规羽毛球 PVC 地胶。
- [ ] 地胶颜色是低饱和青绿色、墨绿色或蓝绿色。
- [ ] 场地线符合羽毛球场视觉，优先白色或黄色。
- [ ] 球馆灯光明亮，地胶有柔和反光。
- [ ] 球拍、拍包、旧羽毛球、运动水壶、毛巾、球网等道具真实自然。
- [ ] 羽毛球拍完整可见：正常椭圆拍框、清楚连续的拍线、笔直拍杆，没有断裂、缺线或畸形。
- [ ] 球拍手柄是橙色手胶，橙色清楚可见，没有被阴影吞掉。
- [ ] 羽毛球是完整白色羽毛球，球托完整，羽毛排列连续清楚，不破碎、不缺羽、不散架、不压扁。
- [ ] 羽毛球放在 PVC 地胶上、场地线旁、拍包旁或球拍旁，不悬浮、不粘在拍线/衣服/手上，不像贴图。
- [ ] 羽毛球符合物理常识：与地面有明确接触点和轻微投影，球托接触地胶或自然侧躺，羽毛冠朝上或斜向展开，比例与球拍、球鞋和场地线一致。
- [ ] 拍包是黑色羽毛球拍包，尼龙材质，拉链和肩带可见，有真实褶皱但不脏乱。
- [ ] 毛巾是暖白色或浅灰色运动毛巾，柔软棉质，有自然折叠。
- [ ] 水具是磨砂白色或黑色翻盖运动水壶，不是矿泉水瓶、一次性塑料瓶、透明塑料瓶、玻璃杯、咖啡杯或保温杯。

## 服装与身体边界

- [ ] 服装适合真实羽毛球运动。
- [ ] 本次服装来自 `outfit-library.md`，或已记录为临时新增服装。
- [ ] 本次服装最近 14 天没有在正式发布中重复使用；如果重复，已经说明原因。
- [ ] 同一篇图文里的多张小羽图片保持同一套服装和道具，至少包括上衣颜色、袖长、下装颜色、鞋袜、拍包、发型和主要道具。
- [ ] 运动裙裤内有安全裤。
- [ ] 鞋子是羽毛球运动鞋或真实室内运动鞋。
- [ ] 腿部线条是健康运动感，不是性感化表达。
- [ ] 没有低胸、湿身感、写真姿势、擦边构图或刻意腿部特写。

## 小红书封面感

- [ ] 画面真实、清爽、阳光，有停留感。
- [ ] 人物服务羽毛球情绪，不是单纯角色美图。
- [ ] 如果是带字封面，文字整体偏大，且有 1 个重点词或短语被明显放大。
- [ ] 如果是内容图文字，字号清楚舒服，不是角落小字；承担情绪标题时也有核心词层级。
- [ ] 可以被理解为运动生活方式内容，而不是网红写真。
- [ ] 首图视觉不会把 YUJI 带向穿搭号、擦边号或 AI 做图号。

## YUJI 品牌 fit

- [ ] 符合“运动热爱痕迹品牌”。
- [ ] 能承载羽毛球人的情绪、记忆、身份感或场景。
- [ ] 前台不需要解释 AI 或虚拟人技术。
- [ ] 如果去掉小羽，画面仍然有羽毛球叙事。
