# Skill：小羽变身参考图提示词

## 触发场景

当 CEO 说“小羽变身”“帮小羽变身”“这张图给小羽变一下”“按这个衣服和 pose 做小羽”时，使用本 Skill。

本 Skill 用于根据 CEO 上传的参考图，提炼图中女生的服装、姿势、构图和氛围，再迁移到 YUJI 固定视觉角色“小羽”，最终输出可直接用于生图的提示词。

## 核心目标

把参考图里的好看部分变成“小羽版本”，而不是复制参考图里的真人。

必须保留：

- 小羽是 YUJI 的女生球友。
- 年龄感 25-30 岁，是成年人。
- 亚洲女生，黑色或深棕色长发，自然披发或低马尾。
- 阳光开朗，眼神明亮，像一进羽毛球馆就被点亮。
- 真实羽毛球运动生活方式，不是虚拟偶像、AI 美女、网红写真或穿搭博主。

## 必读文件

执行前优先参考：

- `docs/yuji-company/visual-system/xiaoyu/prompts.md`
- `docs/yuji-company/visual-system/xiaoyu/outfit-library.md`
- `docs/yuji-company/visual-system/xiaoyu/qa-checklist.md`

## 输入判断

如果 CEO 已上传图片：

1. 先观察参考图中可迁移元素。
2. 不要求 CEO 额外解释，除非图片看不清、人物太小或无法判断服装和姿势。
3. 如果参考图里有多人，只处理最明显的女生主体；不明确时再问 CEO。

如果 CEO 只说“小羽变身”但没有上传图片：

```text
可以，把参考图发我。我会提炼里面的衣服、姿势、构图和氛围，再改成小羽版本的生图提示词。
```

## 分析顺序

### 1. 参考图拆解

只拆可迁移元素：

- 服装：颜色、版型、袖长、下装、鞋袜、层次、配饰。
- 姿势：站姿、坐姿、手部动作、身体重心、头部朝向。
- 构图：全身、半身、近景、侧脸、背影、留白位置。
- 氛围：松弛、清爽、运动感、下班感、刚打完、准备上场。
- 风险：低胸、湿身、擦边、写真化、品牌 Logo、过度网红穿搭、不适合真实打球。

不要描述或复制参考人物的脸、身份、个人特征。

### 2. 小羽迁移

把参考图元素改写成适合小羽和羽毛球场景的版本：

- 不复制真人脸，只保留小羽脸型、五官、发色和气质。
- 不直接照搬不适合打球的衣服；改成适合羽毛球运动的上衣、裙裤、训练短裤、运动鞋。
- 如果参考图衣服有品牌 Logo、明显商标、复杂图案，改成无 Logo、低调运动款。
- 如果参考姿势偏写真、擦边或只为展示身材，改成自然运动生活方式动作。
- 如果小羽是画面主角，必须保持角色、服装和道具风格一致：黑色拍包、完整球拍、橙色球拍手胶、放在地胶上的完整白色羽毛球、运动水壶、毛巾、发饰等都要和当次服装气质匹配。
- 当参考图是偏少女、清爽、粉色、可爱或甜运动风时，道具也要同步改成偏少女但真实可打球的版本，例如奶白、浅灰、浅粉、浅紫、薄荷绿等低饱和颜色的运动水壶、毛巾和小配饰；球拍手胶仍优先保持橙色，拍包仍优先保持黑色。
- 不允许出现“衣服很少女，但拍包、手胶、运动水壶像随机装备或硬核运动男款”的风格断裂。
- 少女感只能用于颜色、材质和小配饰的一致性，不能变成幼态、娃娃感、虚拟偶像感、网红写真或不适合运动的道具。
- 场景默认迁移到真实羽毛球馆，常规羽毛球 PVC 地胶，低饱和墨绿、青绿色或蓝绿色地面。
- 画面必须服务 YUJI 情绪：下班回血、场边休息、最后一局、准备上场、女生球友状态。

### 3. 衣橱判断

优先从 `outfit-library.md` 找相近服装。

如果参考图服装和现有衣橱相近：

- 使用最接近的 `XY-OUTFIT-xx`。
- 输出里写明选用的服装 ID。

如果参考图服装明显有新鲜感，且适合真实羽毛球运动：

- 可以临时生成“参考图改写服装”。
- 必须写清上衣颜色和袖长、下装颜色、鞋袜、发型、拍包、完整球拍、橙色球拍手胶、放在地胶上且符合物理常识的完整白色羽毛球、运动水壶、毛巾和主要道具。
- 必须说明道具风格如何与服装保持一致，尤其是拍包、橙色球拍手胶、运动水壶和毛巾。
- 不自动写入衣橱库，除非 CEO 明确说“加入小羽衣橱”。

## 质量检查

交付前必须自检：

- 是否仍然像小羽，而不是参考图里的另一个女生。
- 服装是否适合真实打羽毛球。
- 拍包、完整球拍、橙色球拍手胶、运动水壶、毛巾和发饰是否与服装气质一致。
- 如果服装偏少女、粉色或清爽可爱，道具是否也偏少女但真实运动，不出现风格断裂。
- 姿势是否自然，不是写真摆拍。
- 场地是否是羽毛球馆 PVC 地胶。
- 图片如果生成出来，是否能服务 YUJI 内容，而不是只像穿搭美图。

## 品牌边界

前台内容不要说“参考图”“Prompt”“AI 改图”“变身流程”。这些只用于内部制作。

对外使用时，小羽仍然是 YUJI 的女生球友，不是 AI 角色展示，不是穿搭博主，也不是低价定制素材。
