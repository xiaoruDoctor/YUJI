# Skill：小红书笔记生成

## 适用场景

把羽毛球情绪观察转成 YUJI 风格的小红书待审核笔记。

## 起步定位

小红书第一阶段先建立“羽毛球人的情绪审美”，不要做成产品号、装备号、教学号、AI 做图号或低价定制号。

YUJI 在小红书上的角色是：羽毛球人愿意投射热爱、记忆、搭子关系和成长痕迹的地方。

## 内容比例

第一阶段按以下比例规划内容：

- 70% 羽毛球情绪瞬间：负责收藏、共鸣和品牌气质。
- 20% 羽毛球人格系列：负责评论、转发和社交传播。
- 10% Player ID / 羽迹时刻 / 战损档案试探：负责验证“我也想拥有一个”的产品信号。

不要反过来。产品露出和 AI 作品只能服务情绪表达，不能成为内容主角。

## 固定产出

- 标题 3 个
- 正文 1 篇
- 封面文案 1 条
- 配图建议 2-3 条
- 标签 8-12 个
- 评论区互动话术 2 条
- 品牌校验

## 内容结构

1. 开头一句击中场景。
2. 中段写具体画面和球友共鸣。
3. 结尾轻轻落到 YUJI：热爱会留下痕迹。
4. 如有产品露出，只占 20%，不要硬卖。

## 起步内容方向

### 羽毛球情绪瞬间

优先写普通球友立刻能代入的具体瞬间：

- 最后一局永远最久。
- 球馆灯亮起来，人就活过来了。
- 有些球不值钱，但就是舍不得扔。
- 成年人的回血方式，是下班后那两小时。
- 退役的不是球拍，是那段时间的自己。

### 羽毛球人格系列

用于提高评论和转发，语气要像球友之间互相调侃：

- 永远说“最后一局”的人。
- 输球后开始研究手胶的人。
- 打混双像打世界赛的人。
- 明明很累，但约球从不拒绝的人。
- 热身半小时，上场三分钟就喊累的人。

### Player ID / 羽迹时刻试探

只展示“这东西很酷”，不要硬卖：

- 球员档案卡。
- 战损球拍档案。
- 搭子纪念卡。
- 年度羽毛球记忆海报。
- 一场球的高光与遗憾记录。

## 标题风格

标题要像球友的心里话，而不是营销口号。

示例方向：

- 最后一局，永远最久
- 有些球不值钱，但舍不得扔
- 球馆灯一亮，人就活过来了
- 退役的不是球拍，是那段时间的自己

## 封面和配图风格

首选“电影感 + 球馆灯光 + 留白 + 成年人的热爱”。

视觉关键词：

- 暖光球馆
- 空场地
- 地板反光
- 背影
- 坐在场边休息
- 系鞋带
- 旧球、旧手胶、断线球拍、磨损球鞋
- 黑色、暖白，少量金色或荧光绿

封面文案尽量一句话，不堆字，不做淘宝风、鸡血风、AI 炫技风或低龄动漫风。

## 互动观察

发布后重点记录以下信号：

- 评论里是否出现“这就是我”“太真实了”“想起我的搭子”。
- 是否有人收藏或转发给球友。
- 是否有人私信问 Player ID、战损档案、记忆海报能不能做。
- 是否有人主动讲自己的旧球、旧拍、搭子、最后一局故事。
