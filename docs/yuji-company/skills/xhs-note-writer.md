# Skill：小红书笔记生成

## 适用场景

把羽毛球情绪观察转成 YUJI 风格的小红书待审核笔记。

## 起步定位

小红书第一阶段先建立“羽毛球人的情绪审美”，不要做成产品号、装备号、教学号、AI 做图号或低价定制号。

YUJI 在小红书上的角色是：羽毛球人愿意投射热爱、记忆、搭子关系和成长痕迹的地方。

## 内容比例

第一阶段按以下比例规划内容：

- 70% 羽毛球情绪瞬间：负责收藏、共鸣和品牌气质。
- 20% 羽毛球人格系列：负责评论、转发和社交传播。
- 10% Player ID / 羽迹时刻 / 战损档案试探：负责验证“我也想拥有一个”的产品信号。

不要反过来。产品露出和 AI 作品只能服务情绪表达，不能成为内容主角。

## 固定产出

- 标题 3 个
- 正文 1 篇
- 封面文案 1 条
- 配图建议 2-3 条
- 每条配图建议对应的 ChatGPT 生图提示词
- 标签 8-12 个
- 评论区互动话术 2 条
- 品牌校验
- 可复用经验判断：本次是否有值得写回 Skill 的新结构、新句型、新禁区或新判断
- 素材库更新判断：本次使用了哪些素材，哪些未使用灵感需要进入选题池

## 内容结构

1. 开头先写一个球友真实会做的小动作、会说的一句话或会遇到的具体场景。
2. 中段写具体画面和球友共鸣，不要先解释洞察。
3. 结尾轻轻落到 YUJI：热爱会留下痕迹。
4. 如有产品露出，只占 20%，不要硬卖。

## 反总结腔规则

小红书正文不能写成“正确地总结了洞察”。如果一个真实打球的人读了没有被带进场景，就算洞察正确，也不是好笔记。

写作时优先使用：

- 小动作：收包时把旧球塞回侧袋、球馆快关灯还在说最后一局、手胶磨黑了还没换。
- 具体感官：灯光、地板反光、喘气声、鞋底摩擦声、拍包、汗印、水喝空了。
- 球友原话：`再来一局？`、`最后一局了`、`这个球还能练两拍`。
- 第一人称或第二人称心里话：`你拍包里有没有过这样一颗球？`

避免使用：

- 先下结论再举例，比如“成年人很多热爱其实没有那么热血”。
- 抽象解释堆叠，比如“情绪价值、身份感、记忆资产、产品机会”。
- 像运营复盘一样的句子，比如“这说明用户对旧物有情感连接”。
- 品牌替用户解释太多，比如一开头就说“YUJI 想记住的是热爱留下的痕迹”。

自检标准：

1. 去掉品牌名后，这篇是否仍像一个真实球友会发的心里话。
2. 前 5 行是否出现具体动作或画面，而不是抽象判断。
3. 正文里是否至少有 2 个羽毛球人熟悉的细节。
4. 读者是否能在评论区自然接一句“我也有过”。

## 用户原话与截图规则

真实用户有感情的话，可以成为小红书笔记的强共情素材，但不能默认直接搬运截图。

优先级：

1. **授权截图**：用户明确同意后，可使用原评论或私信截图，并打码头像、昵称、ID、地区、群名等身份信息。
2. **匿名摘录**：未获得授权时，不放原截图；可以改写成“根据球友讨论整理”的情绪摘录卡。
3. **聚合表达**：多条用户讨论提炼成一句球友心里话，不冒充某个真实用户原话。

截图或摘录使用要求：

- 不把用户原话放成品牌背书或广告证言。
- 不出现可识别个人身份的信息。
- 不截取对方头像、昵称、主页、定位、群聊名称和其他平台水印。
- 不使用带有争议、攻击、隐私、医疗、家庭困境等敏感信息的原截图。
- 如果来自其他平台，正文可写“看到一位球友说过”或“有球友说”，不要制造对方授权代言的感觉。
- 如使用授权截图，应在内部发布稿中记录授权方式和日期，便于后续追溯。

适合呈现的位置：

- 第 2 或第 3 张配图，用作“球友留言 / 情绪摘录卡”。
- 不建议作为首图封面，首图仍优先使用 YUJI 的球馆情绪视觉。

示例标注：

- `来自一位球友的授权留言，已隐去身份信息。`
- `根据公开球友讨论整理，非原文截图。`

## 选题池规则

每次写完小红书笔记后，必须更新或检查 `docs/yuji-company/content-library/topics.md`。

需要写入选题池的内容：

- 当天自动化产出但没有进入主稿的备选标题、切入角度和情绪主题。
- 已经成稿但未来还能复写成系列的主题。
- 评论区、私信或 CEO 想法里出现的可扩展选题。
- 能帮助下次无素材时快速启动的“半成型选题”。

每条选题至少记录：

- 日期
- 状态
- 内容类型
- 来源素材
- 推荐标题
- 核心情绪
- 开头切入
- 评论互动
- 可关联产品信号
- 标签
- 关联笔记

写稿前优先检查：

1. 当天自动化产物。
2. `content-library/materials.md` 中近期未使用素材。
3. `content-library/topics.md` 中 `未使用` 或 `待验证` 选题。

如果选题被写成草稿、待发布或已发布笔记，要回到选题池把状态改为 `已用于笔记`，并补充关联笔记路径。

## 经验回写机制

每次整理小红书笔记后，都要判断是否有内容值得沉淀进 `docs/yuji-company/skills/`，让下一次复用。

值得回写的内容：

- 多次可复用的笔记结构，比如“真实入坑经历 -> 成长转折 -> 情绪出口 -> 想保存的具体瞬间 -> YUJI 轻落点”。
- 明显提升真诚感的表达方式，比如先写个人具体场景，再写品牌使命。
- 新发现的品牌边界，比如第一篇不把 AI 放到前台，不写成创业求支持。
- 可复用的标题句型、封面句型、评论区提问方式。
- 发布后被评论、收藏、私信验证有效的情绪主题。

暂不回写的内容：

- 只适合单篇的个人细节。
- 未经验证、只是临时灵感的口号。
- 会把 YUJI 带偏成装备测评、技术教学、AI 炫技或低价定制的表达。

回写方式：

- 小的写作规则直接补充到本 Skill。
- 品牌语气规则补充到 `yuji-brand-voice.md`。
- 用户情绪洞察补充到 `badminton-emotion-listening.md`。
- 产品信号补充到 `product-insight.md`。
- 未使用素材补充到 `content-library/materials.md`。
- 未使用选题补充到 `content-library/topics.md`。
- 如果形成新的稳定工作流，再新增独立 Skill。

## 创始人故事写法

当 CEO 提供自己的真实经历时，优先把它写成“一个羽毛球人为什么想留下这些瞬间”，不要写成正式品牌公告。

可复用结构：

1. 真实入坑经历：从普通球局、菜鸟局、朋友带入门等具体场景开始。
2. 成长转折：写从快乐打球到想变强、开始早球或加入更高水平球局。
3. 情绪出口：写开心、难过、疲惫时都想去球馆，把羽毛球变成生活出口。
4. 想保存的具体瞬间：写 60 拍、杀球、凌晨球局、下班推门看到球馆灯、和搭子赢下最后一分。
5. YUJI 轻落点：落到“把热爱留下痕迹”，不硬解释产品。

表达替换：

- 不说“我有 AI 工程经验，所以我要用 AI 做这个品牌”，改成“我想用自己擅长的方式，把这些羽毛球记忆沉淀出来”。
- 不说“希望大家支持我的创业”，改成“如果你也有某个只有自己懂的羽毛球瞬间，我们应该会在这里相遇”。
- 不说“我要做一个商业项目”，改成“我想开始认真记录这些时刻”。

## 起步内容方向

### 羽毛球情绪瞬间

优先写普通球友立刻能代入的具体瞬间：

- 最后一局永远最久。
- 球馆灯亮起来，人就活过来了。
- 有些球不值钱，但就是舍不得扔。
- 成年人的回血方式，是下班后那两小时。
- 退役的不是球拍，是那段时间的自己。

### 羽毛球人格系列

用于提高评论和转发，语气要像球友之间互相调侃：

- 永远说“最后一局”的人。
- 输球后开始研究手胶的人。
- 打混双像打世界赛的人。
- 明明很累，但约球从不拒绝的人。
- 热身半小时，上场三分钟就喊累的人。

### Player ID / 羽迹时刻试探

只展示“这东西很酷”，不要硬卖：

- 球员档案卡。
- 战损球拍档案。
- 搭子纪念卡。
- 年度羽毛球记忆海报。
- 一场球的高光与遗憾记录。

## 标题风格

标题要像球友的心里话，而不是营销口号。

示例方向：

- 最后一局，永远最久
- 有些球不值钱，但舍不得扔
- 球馆灯一亮，人就活过来了
- 退役的不是球拍，是那段时间的自己

## 封面和配图风格

首选“电影感 + 球馆灯光 + 留白 + 成年人的热爱”。

视觉关键词：

- 暖光球馆
- 空场地
- 地板反光
- 背影
- 坐在场边休息
- 系鞋带
- 旧球、旧手胶、断线球拍、磨损球鞋
- 黑色、暖白，少量金色或荧光绿

封面成品建议加短文字，是否在生图时直接生成文字，由当次发布目标决定。

封面文字规则：

- 主封面通常需要 6-12 字短文案，帮助信息流里快速识别主题。
- 文字应像球友心里话，不像营销标题。
- 如果 ChatGPT 生图能稳定处理文字，可以直接在提示词里要求生成封面文字，并明确文字内容、位置、字体气质和留白。
- 如果生成文字错字、糊字或排版不稳，再改用干净画面后期手动叠字。
- 如果画面本身信息极强，可以做无字封面测试，但 YUJI 起号阶段优先用“情绪画面 + 短文字”。
- 字体和排版要克制，避免淘宝风大字报、鸡血标语、低龄贴纸和复杂角标。
- 所有带字生图提示词都必须包含这句字体设计要求：`字体参考运动品牌 YUJI Logo 的干刷墨迹质感：粗糙笔触、轻微飞白、边缘有纸感颗粒和沙沙的磨损感，但不要太重，不要像正式书法作品。`
- 怀旧向内容尤其要避免细宋体、端庄书法体、古风字体；封面字要比 Logo 更轻、更清楚，不能喧宾夺主。

怀旧向封面文字要带“时间感”，不能只写物件本身。优先使用：

- 时间钩子：`那天`、`那年`、`最后一局`、`打完那晚`。
- 留存动作：`还在拍包里`、`一直没舍得扔`、`留着的是那天`。
- 轻微遗憾：`那局回不去了`、`它也在场`。
- 人对物说话：`我还把你留在包里`、`我也不舍得扔你`、`你也在场`。

避免只写“旧球 / 你还留着吗”这类单纯提问。它能表达主题，但怀旧感不够。

旧物怀旧主题优先使用第二人称口吻，把旧球、旧拍、旧手胶当成陪过人的对象来写。这样更像人在回看一段时间，而不是运营在描述一个物件。

封面文案尽量一句话，不堆字，不做淘宝风、鸡血风、AI 炫技风或低龄动漫风。

每次给出配图建议时，必须同时给出对应的 ChatGPT 生图提示词。提示词要包含：

- 每条提示词都必须完整、独立、可直接复制使用，不能依赖前文说明或上一条提示词。
- 画幅：优先竖版 `3:4`，适合小红书。
- 风格：写实摄影、电影感、真实运动生活方式。
- 主体：旧球、旧拍、旧手胶、球馆、背影等具体物件或场景。
- 光线：暖光球馆、木地板反光、低饱和、克制留白。
- 构图：近景、低角度、虚化背景、画面留白等。
- 情绪：下班后、散场后、最后一局、疲惫但留恋。
- 文字：如果需要封面字，明确写出 exact text、位置和字体气质，并必须包含 YUJI Logo 干刷墨迹字体标准句；如果不需要，明确写“不要生成文字”。
- 禁止项：不要 logo、不要商品广告感、不要夸张光效、不要清晰人脸。

提示词只作为幕后生图工具使用，前台笔记正文不要提 AI 或生图过程。

## 互动观察

发布后重点记录以下信号：

- 评论里是否出现“这就是我”“太真实了”“想起我的搭子”。
- 是否有人收藏或转发给球友。
- 是否有人私信问 Player ID、战损档案、记忆海报能不能做。
- 是否有人主动讲自己的旧球、旧拍、搭子、最后一局故事。
