# Skill：小羽朋友圈

## 触发场景

当 CEO 说“小羽朋友圈”“做一张小羽朋友圈图”“小羽朋友圈生图”“小羽朋友圈有哪些类型”“按这个主题做小羽朋友圈”或上传参考图并要求转成小羽风格时，使用本 Skill。

本 Skill 用于管理“小羽朋友圈”的生图类型库：先根据 CEO 给的主题、元素或参考图，列出当前可选的朋友圈图类型；等 CEO 选择类型后，再输出该类型的最终生图提示词。

## 定位

“小羽朋友圈”不是微信朋友圈截图，也不是单纯角色美图。

它是一组像小羽自己会发出来的羽毛球生活照片：真实、清爽、有停留感，能持续建立小羽记忆点，同时让用户记住“羽毛球接住了生活里的情绪”。

必须符合：

- 小羽是 YUJI 的女生球友，不是虚拟偶像、AI 美女、穿搭博主或写真模特。
- 画面必须有明确羽毛球元素：球馆、地胶、场地线、球拍、拍包、羽毛球、球筒、球网、鞋带、毛巾或水瓶。
- 每张图服务一个具体情绪或场景，不只展示“小羽好看”。
- 前台内容不讲 AI、Prompt、生图流程或虚拟人技术。

## 工作流

### 1. 输入判断

如果 CEO 只说“小羽朋友圈”，先列出“当前可选类型”，不要直接写完整提示词。

如果 CEO 给了主题或元素，例如“最后一局”“下班回血”“拍包、旧球、橙色手胶”，先把主题映射到最合适的 1-3 个类型，并说明推荐顺序。

如果 CEO 上传参考图：

1. 先拆参考图的构图、动作、氛围和可迁移元素。
2. 不复制参考图人物脸、身份和不可用服装。
3. 如果参考图能归入已有类型，推荐已有类型。
4. 如果参考图形成新类型，先给这个类型命名，并追加到“候选新增类型”里；除非 CEO 明确说“加入小羽朋友圈类型库”，否则不要自动写入本文件。

### 2. 类型选择

列类型时，只给当前已经定义好的类型。每个类型必须包含：

- 类型名
- 适合主题
- 核心画面
- 适合加入的羽毛球元素
- 风险提醒

然后让 CEO 选择一个类型。

### 3. 最终提示词

CEO 选定类型后，输出该类型的最终生图提示词，必须包含：

- 小羽固定角色描述。
- 服装 ID 或完整临时服装描述。
- 场景、构图、动作、情绪。
- 羽毛球元素清单。
- 反向提示词。

默认只产出提示词；只有 CEO 明确说“直接生成图片”“帮我生成图”时，才调用生图工具。

## 当前可选类型

### 类型 01｜出框照片墙

适合主题：

- 小羽朋友圈首图
- 小羽今日球馆状态
- 明亮清爽的羽毛球生活方式图
- 需要强视觉钩子的封面图
- CEO 上传类似“多宫格、照片墙、主体冲出画框”的参考图

核心画面：

- 竖版 3:4。
- 白色相框多宫格照片墙。
- 左侧或中间一张主图最大，小羽和羽毛球道具冲出画框。
- 右侧和底部小图展示同一套服装下的不同羽毛球动作。
- 视觉重点是“多图生活感 + 主体破框的停留感”。

适合加入的羽毛球元素：

- 橙色手胶羽毛球拍。
- 完整白色羽毛球。
- 球筒。
- 黑色拍包。
- 球网。
- 低饱和青绿色、蓝绿色或墨绿色 PVC 地胶。
- 白色或黄色羽毛球场地线。

默认服装建议：

- 清爽明亮：优先 `XY-OUTFIT-07｜浅蓝运动 Polo 白色百褶裙裤`。
- 更日常可爱：可选 `XY-OUTFIT-10｜浅粉上衣白色裙裤`。
- 更运动封面：可选 `XY-OUTFIT-03｜黑色上衣浅灰百褶裙裤`。

风险提醒：

- 不要变成写真拼图。
- 不要只靠脸好看，必须让羽毛球道具成为主信息。
- 出框效果要靠手、球拍、羽毛球、马尾或身体边缘越过白框，不要只做普通拼图。
- 同一张图内所有小羽必须是同一个人、同一套服装、同一套道具风格。

### 类型 01 的最终提示词模板

```text
参考我上传的小羽正脸基础图，保持同一个人的脸型、五官、发色和气质。

生成一张竖版 3:4 小红书封面感真实摄影图片。

画面是白色相框多宫格照片墙。左侧或中间最大主图里，小羽从相框中斜向探出，手、羽毛球拍和白色羽毛球明显突破画框，形成强烈 3D 出框效果。右侧和底部小图里是同一个小羽、同一套服装、同一场馆光线，分别呈现拿拍微笑、抛羽毛球准备发球、拿球筒、坐在拍包旁休息等羽毛球瞬间。

人物是 YUJI 的女生球友小羽，亚洲女生，25-30 岁，黑色或深棕色长发低马尾，眼神明亮，笑容自然，像一进羽毛球馆就被点亮。她穿【服装 ID 对应完整服装描述】。

场景是真实室内羽毛球馆，低饱和蓝绿色、青绿色或墨绿色 PVC 地胶，白色或黄色羽毛球场地线，球网局部可见，地胶有柔和反光。道具有橙色手胶羽毛球拍、完整白色羽毛球、球筒、黑色拍包。

整体真实、清爽、阳光，有小红书运动生活方式封面感。不要文字，不要水印。
```

反向提示词：

```text
不要复制参考图真人的脸，不要水果，不要草莓，不要篮子，不要户外蓝天，不要蕾丝头巾，不要波点裙，不要低胸，不要湿身感，不要写真姿势，不要擦边，不要刻意腿部特写，不要品牌 Logo，不要篮球馆，不要木地板，不要破损畸形羽毛球，不要多余手指，不要扭曲手部，不要重复脸，不要同图服装不一致，不要虚拟偶像感，不要 AI 科技感。
```

## 后续扩展规则

每新增一种小羽朋友圈类型，必须补充：

- 类型编号和名称。
- 适合主题。
- 核心画面。
- 适合加入的羽毛球元素。
- 默认服装建议。
- 风险提醒。
- 最终提示词模板。

命名要短，便于 CEO 选择，例如：

- 类型 02｜场边随手拍
- 类型 03｜最后一局散场照
- 类型 04｜拍包静物合照
- 类型 05｜球馆门口打卡照

新增类型前先判断：它是否提供了新的构图或情绪，不只是换一套衣服。如果只是服装变化，不新增类型。
