# Skill：羽毛球情绪采集

## 适用场景

当需要为 YUJI 寻找小红书选题、用户情绪、产品洞察或市场观察时使用。

当进行每周“羽毛球情绪价值雷达”研究会前，也使用本 Skill 采集和筛选真实表达。

## 采集目标

寻找羽毛球人的真实表达，而不是只找新闻和装备信息。

优先捕捉：

- 为什么舍不得扔某个球、手胶、鞋或球拍
- 最后一局、搭子、混双、输赢、爆线、战损等共鸣
- 打球后的情绪恢复、压力释放和身份认同
- 用户吐槽现有工具、记录、复盘、定制商品的不满
- 能转成 Player ID、羽迹时刻、战损档案的故事

每周研究会优先滚动观察：

- 身份认同
- 回忆与纪念
- 社交与归属感
- 荣誉感与身份系统
- 仪式感行为
- DIY 与个性化
- 情绪恢复与精神价值

## 全网采集执行规则

默认不要把 Reddit 当成唯一主来源。YUJI 的情绪采集要同时看国内语感、海外结构和公开网页的补充证据。

### 来源顺序

1. 先看国内可访问来源，找今天最贴近小红书语境的真实表达。
2. 再看海外社区，补充更完整的情绪结构、关系细节和长期主题。
3. 最后用公开网页、行业站、论坛长帖和搜索扩展结果做交叉验证或补盲。

### 国内可访问来源优先池

- 爱羽客圈子、评论区、自述文章
- 中羽在线论坛、故事贴、公开讨论
- 知乎问答、回答区、评论区
- B站视频评论区、动态评论区
- 虎扑公开讨论
- 即刻公开讨论
- 百度贴吧公开贴页
- 豆瓣公开小组贴页
- 微信公众号公开文章和评论区
- 小红书公开可访问页、搜索结果摘要、二次转载或站外讨论页

### 海外来源优先池

- Reddit
- X
- YouTube 评论区
- Facebook 羽毛球社群公开讨论
- Discord 公开内容摘要
- BadmintonCentral Forum
- Quora

### 搜索扩展来源

- 搜索引擎 `site:` 检索结果
- 球馆、赛事、社群和个人博客的公开自述
- 公开采访、评论区、聚合页和论坛长文

### 来源配比

- 单次早报或晚报，不要让全部 3-5 条观察都来自同一平台。
- 优先做到“国内来源 + 海外来源 + 搜索扩展来源”同时出现。
- 如果当天国内来源弱，只能拿到搜索摘要或转载页，必须在来源说明里写清楚，不把弱证据包装成完整一手表达。

## 输出格式

每条观察包含：

- 原始表达摘要
- 来源平台或链接
- 来源说明（原帖 / 评论区 / 搜索摘要 / 长文 / 聚合页）
- 情绪主题
- 情绪本质
- 是否存在消费行为或付费线索
- 可转成的小红书表达
- 可转成的产品机会

## 素材入库规则

每次完成情绪采集后，必须判断哪些素材值得进入 `docs/yuji-company/content-library/materials.md`。

必须入库的素材：

- 当天主笔记已经使用、后续还可能复用的高质量素材。
- 当天没有使用，但有清晰情绪主题和复用潜力的素材。
- 能转成 Player ID、羽迹时刻、战损档案、搭子纪念卡或运动记忆海报的真实表达。
- 评论、私信或 CEO 临时想法中出现的具体场景。

入库时保留：

- 日期
- 状态
- 来源
- 原始表达摘要
- 情绪主题
- 可转成的小红书表达
- 可转成的产品机会
- 标签
- 关联笔记

状态建议使用：

- `未使用`
- `已用于笔记`
- `待验证`
- `验证中`
- `已沉淀`
- `暂缓`

## 判断标准

优先选“普通球友能立刻代入”的内容。少选宏大行业分析，多选具体瞬间。

额外判断：

- 国内平台的语感是否更接近小红书受众。
- 海外平台是否提供了更完整的情绪结构，而不是只提供热词。
- 这一条是强证据、弱证据，还是只能当线索继续追。

如果素材用于每周研究会，还要额外判断：

- 是否能支撑下周主内容情绪。
- 是否能支撑一个低成本 MVP 假设。
- 是否值得 CEO 参与讨论。
