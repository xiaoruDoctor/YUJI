# Skill：羽毛球情绪采集

## 适用场景

当需要为 YUJI 寻找小红书选题、用户情绪、产品洞察或市场观察时使用。

## 采集目标

寻找羽毛球人的真实表达，而不是只找新闻和装备信息。

优先捕捉：

- 为什么舍不得扔某个球、手胶、鞋或球拍
- 最后一局、搭子、混双、输赢、爆线、战损等共鸣
- 打球后的情绪恢复、压力释放和身份认同
- 用户吐槽现有工具、记录、复盘、定制商品的不满
- 能转成 Player ID、羽迹时刻、战损档案的故事

## 输出格式

每条观察包含：

- 原始表达摘要
- 来源平台或链接
- 情绪主题
- 可转成的小红书表达
- 可转成的产品机会

## 判断标准

优先选“普通球友能立刻代入”的内容。少选宏大行业分析，多选具体瞬间。
