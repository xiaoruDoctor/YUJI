# Skill：小羽轻图文

## 适用场景

当 CEO 提到“小羽轻图文”“轻型图文”“轻图文”“今日份小羽”“小羽 OOTD”“小羽日常”“一张图配一两句话”“用小羽照片发个小红书”时，使用本 Skill。

本 Skill 用于生成低成本、高频、小红书友好的小羽轻图文：用 1-2 张图和 1-4 行情绪文案，让账号持续建立小羽记忆点和羽毛球情绪审美。

## 定位

小羽轻图文不是普通穿搭笔记，也不是角色美图。

它的作用是：

- 给 YUJI 账号增加轻量更新频率。
- 让小羽作为 YUJI 形象大使持续出现。
- 用一个具体动作或一句心里话测试用户是否停留、收藏、关注或评论。
- 为后续重图文、用户故事和产品试探提供轻信号。

## 内容边界

必须符合：

- 小羽是 YUJI 的女生球友，不是穿搭博主、虚拟偶像或 AI 美女。
- 每篇只讲一个小动作、一个球馆瞬间或一句情绪。
- 画面必须有明确羽毛球线索：球馆、地胶、场地线、球拍、拍包、羽毛球、鞋带、毛巾或水瓶。
- 正文短，但不能空。必须像真实球友发照片时顺手写下的心里话。
- 前台不讲 AI、Prompt、生图流程或虚拟人技术。

禁止方向：

- 纯 OOTD 穿搭展示。
- 写真感、擦边感、刻意腿部特写。
- 装备测评、技术教学、商品种草。
- 空泛文艺句，没有羽毛球场景。
- 让用户只记住“小羽好看”，而不是“这个球馆瞬间我懂”。

## 类型

默认从以下类型中选择一个，不要一篇塞多个主题：

| 类型 | 适合内容 | 示例主题 |
|-|-|-|
| 小羽 OOTD | 服装有新鲜感，但必须绑定动作和情绪 | 上场前先把自己系好 |
| 球馆 5 分钟 | 场边等待、系鞋带、喝水、整理拍包 | 坐下来的那几分钟，人才慢慢回来 |
| 下班回血 | 下班到球馆、推门、换鞋、被灯光点亮 | 推开球馆门，今天才属于我 |
| 最后一局 | 散场、收包、回头看场地 | 最后一局说出口的时候，谁都不想走 |
| 回场日常 | 停了一段时间后回来 | 停了一周，才知道自己有多想这片地胶 |
| 固定球局 | 熟悉的时间、搭子和场地 | 成年人的归属感，有时候就是固定球局 |

## 输入判断

如果 CEO 已给图片：

1. 先判断图片是否适合做轻图文。
2. 如果图片已经是一张可用成图，默认不要重新生成整张图；只基于 CEO 给的原图做封面文字添加、轻微调色、轻微锐化、局部留白优化或小范围瑕疵修复。
3. 如果适合，直接围绕图片里的动作、服装、场景和情绪生成笔记，并给“基于原图编辑”的封面优化提示词。
4. 如果图片只是参考图、没有小羽、或 CEO 明确说“重新生成”，才给重新生图提示词。
5. 如果图片更适合“变身生图提示词”，可同时提示使用 `xiaoyu-transform-prompt.md`。

如果 CEO 只给主题：

1. 根据主题选择轻图文类型。
2. 给出建议画面和可复制生图提示词。

如果主题不明确：

只问一个问题：

```text
这条轻图文你想更偏哪种：小羽 OOTD、下班回血、场边休息、最后一局，还是回场日常？
```

## 正文规则

轻图文正文控制在 1-4 行。

优先结构：

```text
一个具体画面

一句球友心里话
```

或：

```text
一句情绪标题

一个具体动作
```

可复用句型：

```text
鞋带系好的那一刻
今天才真的下班了
```

```text
有时候不是想变强

只是想去球馆待一会儿
把自己慢慢捡回来
```

```text
最后一局说出口的时候
其实谁都不想走
```

```text
停了一周才知道
自己有多想这片地胶
```

正文和一键复制块里，句末默认不加中文句号 `。`。

## 图片规则

轻图文可以只发 1 张图；需要增强叙事时发 2 张。

推荐：

- 1 张 CEO 已给的小羽成图：优先保留原图主体、姿势、场景和构图，只加封面字和轻微优化。
- 1 张羽毛球相关静物图：球拍、拍包、鞋、毛巾、水瓶、旧球或地胶。

不建议：

- 连续多张小羽正脸美图。
- 只有人物没有羽毛球线索。
- 服装每张都变，导致小羽像不同人。

同一篇多图必须锁定同一套服装和道具。小羽服装优先从 `docs/yuji-company/visual-system/xiaoyu/outfit-library.md` 选择最近 14 天未使用过的一套。

## 封面优化提示词规则

轻图文的默认路径是：CEO 给一张完整成图，Codex 产出小红书文案，并给“把封面文字加到原图上”的编辑提示词。

当 CEO 已经给成图时，提示词必须明确：

- 基于我上传的原图编辑，不要重新生成新人物、新球馆或新构图。
- 保留原图人物、姿势、服装、球拍、毛巾、水瓶、地胶、球网和整体光线。
- 只做轻微优化：加入封面文字、增强文字可读性、轻微调整亮度和对比度、保持真实摄影质感。
- 不改变人物脸、五官、发型、身体比例、服装款式和动作。
- 不裁掉关键羽毛球元素，例如球拍、鞋带、场地线、球网和地胶。
- 如果文字区域不够，可以在原图留白处轻微压暗背景或加半透明柔和阴影，但不要加大色块、贴纸、边框或营销海报装饰。

标准编辑提示词结构：

```text
基于我上传的原图进行封面编辑，不要重新生成整张图，不要改变人物、脸、动作、服装、球馆和构图。

请在原图【位置】添加两行清晰中文封面文字：
第一行“...”
第二行“...”

字体参考运动品牌 YUJI Logo 的干刷墨迹质感：粗糙笔触、轻微飞白、边缘有纸感颗粒和沙沙的磨损感，但不要太重，不要像正式书法作品。文字颜色使用温暖米白色或淡奶油色，可以加轻微柔和阴影，保证手机信息流里一眼可读。

只做轻微画面优化：略微提升整体清晰度和对比度，让球馆地胶、人物和文字更清楚；保持真实摄影感和原图安静克制的球馆氛围。

不要重画人物，不要换脸，不要换衣服，不要改变姿势，不要改变场地，不要添加品牌 Logo，不要加贴纸，不要做营销海报感，不要过度磨皮，不要过度调色。
```

## 重新生图提示词规则

只有在 CEO 没有给图、原图不可用，或 CEO 明确要求“重新生成封面图”时，才输出从零生成图片的提示词。

如果需要生成图片，提示词必须包含：

- 参考小羽基础图，保持同一个人的脸型、五官、发色和气质。
- 小羽是 YUJI 的女生球友，亚洲女生，25-30 岁，真实羽毛球运动生活方式气质。
- 具体动作：系鞋带、整理拍包、喝水、站在边线外、收包、看向球场等。
- 完整服装：上衣颜色和袖长、下装颜色、鞋袜、发型、拍包和主要道具。
- 真实羽毛球馆，常规羽毛球 PVC 地胶，低饱和墨绿、青绿色或蓝绿色地面，白色或黄色场地线。
- 禁止项：不要低胸、湿身、写真姿势、擦边、刻意腿部特写、品牌 Logo、商品广告感、篮球馆、木地板、AI 科技感。

如果封面需要文字，必须写明 exact text，并包含：

`字体参考运动品牌 YUJI Logo 的干刷墨迹质感：粗糙笔触、轻微飞白、边缘有纸感颗粒和沙沙的磨损感，但不要太重，不要像正式书法作品。`

## 观察指标

轻图文发布后重点看：

- 是否带来关注。
- 是否有人评论“这就是我”“我也这样”“小羽好像我球馆里的朋友”。
- 是否有人开始记住“小羽”。
- 是否收藏或转发给球友。
- 是否评论自己的固定动作、固定球局、下班回血或最后一局。

轻图文不以单篇长评论为唯一标准。它更重要的是稳定角色识别、低成本测试情绪和提高账号亲近感。

## 复盘与回写

连续 7 条小羽轻图文后，需要复盘：

- 哪类动作最有停留感。
- 哪类一句话最容易被收藏。
- 小羽是否被用户记住。
- 是否有偏离：穿搭化、写真化、颜值化或脱离羽毛球。

有效结构回写到本 Skill；强产品信号回写到 `product-insight.md`；可复用选题回写到 `content-library/topics.md`。
