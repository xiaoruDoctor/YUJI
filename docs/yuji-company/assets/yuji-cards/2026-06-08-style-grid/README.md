# 羽迹卡九宫格风格母版｜2026-06-08

## 资产定位

这张九宫格是羽迹卡第一版风格母版，用于小红书图文里让粉丝选择自己想要的羽迹卡类型。它不是单次配图，而是后续羽迹卡产品化、内测和商业化的起点资产。

- 关联笔记：`docs/yuji-company/xhs-notes/published/2026-06-09-005-an-se-young-yuji-card.md`
- 图片文件：`yuji-card-style-grid-v1.jpg`
- 当前用途：安洗莹 2026 六冠节点笔记的补充图，用于引导粉丝评论风格偏好。
- 验证目标：观察粉丝是否会主动选择 A-I 中某一种风格，并愿意提供球龄、球风、故事、照片和一句心里话。

## 总体收集原则

当粉丝评论“想要某某风格”时，不要马上承诺批量制作，也不要在评论区索要隐私信息。先把它当作产品信号，轻量收集资料。

公开评论区可以引导：

```text
你选的是 X 风格。我先收一版内测素材：球龄 / 球风 / 最想留下的一场球 / 一句心里话。照片和隐私信息可以私信，不用发在评论区。
```

私信中可收集的通用资料：

| 字段 | 是否必填 | 说明 |
|-|-|-|
| 选择风格 | 必填 | A-I 中选择一个，最多允许再补一个备选。 |
| 称呼或昵称 | 必填 | 可用昵称，不要求真实姓名。 |
| 球龄 | 必填 | 例如 3 个月、1 年、6 年。 |
| 球风 | 必填 | 例如进攻型、防守型、混双、爱救球、新手上头型、下班回血型。 |
| 想留下的一场球 | 必填 | 时间、地点、和谁、为什么忘不了，写清楚一个具体瞬间。 |
| 一句心里话 | 必填 | 卡面最重要的一句话，最好是球友自己的话。 |
| 照片 | 选填 | 本人背影、球拍、球馆、旧球、搭子合照均可。涉及他人必须获得对方同意。 |
| 是否允许公开展示 | 必填 | 明确“可公开 / 只内部试做 / 需要打码后公开”。 |

不收集：

- 身份证、手机号、住址等敏感信息。
- 真实指纹、精确定位、未授权他人照片。
- 未获得授权的赛事摄影图或他人肖像。

## 生图提示词结构

后续真正为粉丝生成羽迹卡时，先用通用基础提示词，再拼接对应 A-I 风格模块。不要直接复用粉丝隐私信息，公开样稿要先打码或改写。

### 通用基础提示词

```text
生成一张竖版 4:5 羽毛球故事海报，属于 YUJI 运动热爱痕迹品牌。画面要像一张可保存的羽迹卡，不是商品广告，不是低价定制模板。整体气质：电影感、克制留白、真实羽毛球生活方式、暖光球馆、低饱和、高级运动品牌视觉。

场地必须是常规羽毛球 PVC 地胶，低饱和墨绿、青绿色或蓝绿色，白色或黄色羽毛球场地线，地胶有柔和反光和轻微汗印。避免篮球馆、木地板、红色跑道、灰色水泥地、错乱场地线。

人物尽量不露脸，优先背影、侧影、手部、腿部或局部动作。如果出现两个人，关系是球友或搭子，不要情侣写真感。女生球友要真实运动生活方式，不要网红摆拍。

卡面文字控制在少量清晰中文，保留编号、标题、短句和必要字段。字体参考运动品牌 YUJI Logo 的干刷墨迹质感：粗糙笔触、轻微飞白、边缘有纸感颗粒和沙沙的磨损感，但不要太重，不要像正式书法作品。

不要出现 AI、Prompt、定制接单、低价、商城、联系方式等字样。
```

### A 冠军档案模块

```text
风格选择：A 冠军档案。画面是一位不露脸羽毛球玩家的背影，手持球拍站在暖光球馆聚光灯下，黑金配色，档案编号、荣誉节点、胜利关键词以克制小字排版。核心情绪是“这是我自己的高光局”，即使不是职业比赛，也像认真保存一场属于自己的胜利。
```

### B 球馆电影模块

```text
风格选择：B 球馆电影。画面是球馆散场后的暖光、地胶反光和一个收包或走出球馆的背影。卡面像电影海报，文字很少，突出那一天的时间、地点和一句安静的心里话。核心情绪是“那场球结束了，但我还记得”。
```

### C Player ID 模块

```text
风格选择：C Player ID。画面是一张现代运动身份卡，包含昵称、球龄、惯用手、球风、常去球馆、搭子关系、专属称号和一句代表自己的话。卡面可以有虚构编号、条形码和虚构纹理，但不要使用真实指纹或敏感身份信息。
```

### D 战损球拍模块

### E 热血漫画模块

```text
风格选择：E 热血漫画。画面是关键分瞬间，羽毛球玩家挥拍、救球或扣杀，使用漫画分镜、速度线、颗粒纸感和强对比构图。卡面突出关键比分和一句短口号。整体热血但不要幼稚，保留真实羽毛球运动质感。
```

### F 极简杂志模块

```text
风格选择：F 极简杂志。画面像高级运动杂志封面，大面积留白，只有球拍局部、人物侧影、手部动作或一只旧球。文字极少，使用一个主题词和一句短句。核心情绪是克制、自律、专注，不要堆满信息。
```

### G 搭子双人模块

```text
风格选择：G 搭子双人。画面是一位女生和一位男生球搭子，或两位真实球友，以背影、并肩站立、递水、收包、看场地等方式出现。关系真实、自然、有默契，不要暧昧，不要情侣写真。卡面记录两个人常说的一句话和最难忘的一局。
```

### H 最后一局模块

```text
风格选择：H 最后一局。画面是灯快关的羽毛球馆，地上有一支球拍、一颗旧羽毛球或散落的球包，人物可不出现。整体低饱和、安静、有告别感。卡面记录日期、地点和一句“这局我会记很久”的话。
```

### I 新手成长模块

```text
风格选择：I 新手成长。画面是一位新手球友在场边系鞋带、握拍练习或看训练小纸条，旁边有旧球、水瓶、拍包和练习痕迹。可以优先使用女生新手视角。核心情绪是“第一天不会打，后来真的开始上头了”，温柔记录成长，不嘲笑新手。
```

## 内部处理流程

1. 评论出现 A-I 选择后，先记录在产品信号池或发布反馈里。
2. 公开回复只引导 4 个通用信息，不在评论区索要照片和隐私。
3. 私信收集资料后，判断是否符合 YUJI：是否有具体场景、具体关系、具体旧物或具体情绪。
4. 优先试做故事最具体的 3-5 个，不按先到先得机械执行。
5. 生成样稿前先确认公开权限：可公开、匿名公开、只内部试做。
6. 样稿完成后记录：选择风格、用户提供信息、生成提示词版本、是否公开、用户反馈。

## 判断指标

这张九宫格资产后续要重点观察：

- 评论里哪个编号被选得最多。
- 是否有人主动讲自己的球局、搭子、旧拍、最后一局或新手故事。
- 是否有人愿意私信提供照片和一句心里话。
- 是否出现“能不能帮我做”“我也想要一张”“想给搭子做一个”。
- 哪些风格更容易带来真实故事，而不只是“好看”。

## 品牌边界

- 羽迹卡是运动记忆海报，不是低价定制服务。
- 先验证情绪和故事密度，再考虑产品化和商业化。
- 前台不讲 AI 和生图流程，AI 只是幕后能力。
- 公开展示必须尊重用户授权和他人肖像。
- 职业运动员卡必须校对事实，不写成品牌背书。
