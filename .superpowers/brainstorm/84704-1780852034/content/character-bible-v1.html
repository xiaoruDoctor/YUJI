YUJI 虚拟女生球友：角色设计稿 v1

这是进入正式角色圣经前的确认稿。重点是：脸清晰、形象稳定、能放进不同球馆场景，但不变成虚拟偶像或擦边流量。

三种建设路径

A

纯真人感角色

最像真实小红书博主，封面效率高；但容易被误解成真人账号，也更容易撞脸或陷入肖像风险。

B

强 IP 女主角

辨识度最高，适合长期视频栏目；但会把 YUJI 推向虚拟偶像，弱化羽毛球人的真实情绪。

C

推荐：真实纪录片 + 轻角色识别

脸清晰、五官稳定、气质有记忆点；但拍法、服装和情绪始终像真实球友。这是最适合 YUJI 的长期路线。

角色核心

角色定位：YUJI 的虚拟女生球友，不是代言人，也不是偶像。她是读者进入“羽毛球情绪出口”的第一视角。

年龄感：25-30 岁，刚进入稳定工作与生活压力并存的阶段。

第一气质：安静、有点累，但眼神里还有一局。

关系感：可以一个人坐在场边，也可以和男生/女生搭子同场，但不营造情侣感。

脸部规则

脸：正脸清晰，亚洲女生，真实运动生活方式，不走网红精修感。

五官：清淡、耐看、略有距离感；眼神比笑容重要。

表情：少大笑，常见轻微疲惫、低头、发呆、呼吸刚缓下来。

一致性：固定脸型、眼距、鼻梁、嘴唇、发色发长和右手持拍习惯，后续正脸/侧脸/背影统一。

身体与服装边界

身材：健康、真实、长期运动的人，不强调性感曲线。

服装：黑/白/深灰运动 T 恤，短裤或运动裙可用，但必须适合真实打球。

禁区：低胸、过短摆拍、写真姿势、刻意腿部特写、湿身感、网红妆。

可识别物：黑色拍包、磨损球鞋、旧手胶、低饱和绿色球馆地胶。

首批场景

1. 场边坐着，低头握拍，像刚输完一局或刚喘过来。

2. 下班后到球馆，拍包还没完全放下，灯光很亮但人很安静。

3. 最后一局后收包，旧球、手胶、地胶反光在画面里。

4. 混双训练间隙，和搭子只保留运动默契，不写暧昧。

一句角色锚点

她不是来证明自己多会打。她只是每次快被生活耗空的时候，还会去球馆再打一局。
